# Getting started
### Prerequisites
* [MongoDB](https://docs.mongodb.com/manual/installation/)
* [Yarn](https://yarnpkg.com/en/docs/install)

### Setup
* `sh setup.sh`

### Frontend usage
* `cd frontend`
* `yarn start`

### Backend usage
* `cd backend`
* Make sure MongoDB is running:
  * `mongod` 
* `npm start`
